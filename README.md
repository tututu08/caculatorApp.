```markdown
# CalculatorApp

iOS 계산기 앱

## 기능 요약

- 기본 사칙연산 (`+`, `-`, `*`, `/`) 수행
- `AC`로 초기화
- `=` 버튼으로 계산 실행
- 연산자 중복 방지
- 연산자 뒤에 `0`이 연속 입력되는 경우 무시
- 연산자 뒤에 숫자가 오면 `0`을 덮어쓰도록 처리

## UI 구성

- 상단: 결과 표시용 `UILabel`
- 하단: 4x4 버튼 레이아웃 (`UIStackView` 활용)
- 버튼 종류:
  - 숫자: 0~9
  - 연산자: `+`, `-`, `*`, `/`
  - 기타: `AC`, `=`

## 예외 처리

- 연산자로 끝나는 상태에서 `=` 입력 시 무시
- 연산자 중복 입력 시 기존 연산자 제거 후 새 연산자 입력
- 연산자 뒤에 `0`을 입력할 경우 무시
- 연산자 뒤에 숫자 입력 시, `0`을 덮어쓰기
- 전체가 `"0"`일 경우 숫자 입력 시 덮어쓰기

## 파일 구성

- `ViewController.swift`: UI 구성 및 버튼 동작 정의
- `SnapKit`: 오토레이아웃 설정에 사용

## 향후 개선 예정

- 소수점 처리
- 괄호 입력 처리
- 백스페이스 기능 추가
- 연산 우선순위 커스터마이징

```
